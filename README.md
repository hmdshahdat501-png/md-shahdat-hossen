## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)

## ✨ Key Features

- **Modern Editorial Design:** High-impact typography and grid layouts inspired by premium digital journals.
- **Fluid Interactions:** Smooth page transitions, micro-animations, and a custom interactive cursor using Framer Motion.
- **Tech Stack Visualization:** Dynamic display of core competencies including Docker, Kubernetes, Socket.IO, and Prisma.
- **Responsive Architecture:** Fully optimized for all device sizes with mobile-first precision.
- **Dark/Light Mode:** Seamless theme switching with persistent user preference.

## 🚥 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shakil218/md-shakil-islam-portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Go to [https://md-shakil-islam-portfolio-dev.vercel.app](https://md-shakil-islam-portfolio-dev.vercel.app)
